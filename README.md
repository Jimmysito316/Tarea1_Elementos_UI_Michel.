DESCRIPCIÓN DE LA APP:
Esta aplicación es un catálogo modular de los elementos fundamentales de la interfaz de usuario (UI) en Android. El objetivo es demostrar el dominio de los componentes básicos mediante el uso de Fragments para una navegación eficiente y organizada.

COMPONENTES INCLUIDOS:
- Manejo de entradas de texto (Campos).
- Eventos de interacción (Botones).
- Selección de opciones múltiples y únicas (Selección).
- Visualización de datos dinámicos (Listas).

INSTRUCCIONES DE USO:
1. Al iniciar la aplicación, utilice la barra de navegación inferior para cambiar entre las 4 secciones principales.
2. En cada sección, interactúe con los elementos para validar las respuestas del sistema (mensajes Toast, cambios de estado o ingreso de datos).
3. Para regresar o cambiar de categoría, simplemente presione el botón correspondiente en el menú.
